- 应用服务接口按读写分离拆分为继承 ApplicationCmdService / ApplicationQueryService 的两个独立接口，而非单一聚合接口。
- 所有入参 DTO 继承 BaseDto 并重写 check() 进行参数校验，返回 ResultDO<Void>；出参 DTO 通过 ResultDO<T> 统一包装业务结果。
- 包含大对象字段（byte[] content）的 DTO 使用 @ToString(exclude = "content") 避免序列化溢出。
- 跨响应复用的纯元数据模型放入 model/ 包，具体请求/响应分别放在 req/ 与 res/ 包下。
