本仓库为单 Maven 聚合工程，pom.xml 通过 properties 集中声明 Java 25、MapStruct 1.6.3、MyBatis-Flex 1.11.8、Sa-Token 1.45.0、springdoc 3.x、AWS S3 SDK v2 等版本，并由 maven-compiler-plugin 统一注册 lombok/mapstruct/mybatis-flex 注解处理器；所有业务子模块（auth_domain、system_user、system_role、system_dict、system_file、system_log、system_online）均依赖 common_shared 提供的 DDD 基类、统一结果/异常与安全上下文，从而形成「公共基类 → 领域子域」的单向依赖树。

运行时基础设施由 docker-compose.yaml 提供 PostgreSQL 17 与 Redis 7，应用启动时 Flyway 自动执行 db/migration 完成建表；Sa-Token 会话与在线用户数据落 Redis，文件上传走 AWS S3 协议客户端，日志通过 Spring 事件异步写入数据库。

顶层还暴露 .env.example、mvnw/mvnw.cmd、rename-project.sh 等脚手架脚本，用于快速初始化与重命名项目。
