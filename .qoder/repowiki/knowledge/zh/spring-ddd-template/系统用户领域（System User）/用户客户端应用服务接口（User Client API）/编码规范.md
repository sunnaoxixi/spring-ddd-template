- 应用服务按读写分离拆分为两个接口：写操作继承 ApplicationCmdService，读操作继承 ApplicationQueryService，方法签名统一为 ResultDO<ResponseDTO> xxx(RequestDTO) 形式。
- 请求/响应 DTO 成对放置在 req/ 与 res/ 子包中，命名遵循 XxxRequestDTO / XxxResponseDTO 约定，并通过 Assembler 与内部模型解耦。
- 所有 DTO 继承 BaseDto，包含 serialVersionUID，使用 Lombok @Getter/@Setter/@ToString，并在 check() 中基于 ErrorCodeEnum.PARAM_ERROR 返回校验结果。
- 对外枚举（如 UserStatusEnum）独立于内部 domain 枚举定义，并提供 getByCode 静态工厂方法，避免 client 层反向依赖内部模型。
