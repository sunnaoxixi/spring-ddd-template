该仓库是一个 Spring Boot DDD 后端脚手架项目，不包含任何前端代码或样式资源。经检查：
- 未发现任何 CSS、SCSS、LESS、Tailwind 配置文件
- 未发现 Vue/React/Angular 等前端框架源码
- `src/main/resources/static` 与 `public` 目录不存在
- 项目中仅通过 OpenAPI/Swagger 提供接口文档（`/swagger-ui.html`），用于前后端协作，但本身不是前端样式系统
- 搜索关键词 `style/theme/css/scss/tailwind/bootstrap/antd/element-ui/material/design-token` 仅匹配到 Sa-Token 的 token 样式配置和 S3 路径风格配置，与 UI 无关

因此本仓库不涉及前端样式主题设计，属于纯后端项目，`frontend_style` 类别不适用。
